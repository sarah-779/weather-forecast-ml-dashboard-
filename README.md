# 🌦️ Weather Forecast ML Dashboard

A Machine Learning powered Streamlit app for weather analysis and rain prediction.

## 🚀 Features
- Rain prediction using ML (Random Forest)
- Temperature & humidity analytics
- Interactive Streamlit UI
- Charts (Plotly)
- AI-generated insights

## 🧠 ML Model
Random Forest Classifier trained on weather dataset.

## ▶️ Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
